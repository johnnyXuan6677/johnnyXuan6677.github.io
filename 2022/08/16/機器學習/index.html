      Machine learning glossary(術語)

  • Sample or input or data point
    • 就是inputdata和統計學上的意思是一樣的
    • 樣本
  • Prediction or output
    • 基於過去的經驗所推導出來的值
  • Target or label
    • 目標，判斷，標籤
  • Loss value or prediction error
    • 我所產生出來的預測和實際答案的差異就是損失函數
    • loss function愈低model越好
  • Classes
    • 用cluster做label
  • Binary classification
    • 二元分類
  • Multi-class classification
    • 複數類別的可能性是多少，進而推演出最有可能是甚麼類別
    • 是貓的可能性是?，狗的可能性是?
    • 一個class底下也有可能會有多個標籤
      • 餐廳-西式中式美式
  • Scalar regression
    • 常數回歸
    • 輸入一系列的數字進而做回歸
  • Vector regression
    • 至少是至少以兩個數據資料為單位
      • 多種數據做判斷
    • 向量回歸
  • Batch(批量)
    • 一張一張訓練做過慢，所以用批量(可能以64的資料數為一個單位)的方式做訓練會更加快速
  • Epoch
    • 不是批量
    • 我不管你批量多大，只管要重新訓練幾次
    • 為了讓你的loss function可以確認最佳化

      Evaluating machine learning models

  • 訓練出來model的準確程度評估
  • 挑整神經元的數目
    • dropout
      • 打掉原來的model
  • 訓練集(Training dataset)：
    • 學習樣本數據集，通過匹配一些參數來創建一個分類器。創建一種分類的方式，主要是用來訓練模型的。
  • 驗證集(Validation dataset)：
    • 驗證集基本是在每個epoch完成後，用來測試一下當前模型的準確率，並且調整分類器或演算法的參數以及超參數。
  • 測試集(Testing dataset)：
    • 主要是測試訓練好的模型的分辨能力（識別率等）
  • parameters and hyper parameter
    • 參數(parameter)通常是在模型訓練的過程中，我們根據訓練集資料自動得到的。
    • 超參數(hyperparameter)通常是在模型訓練前，我們手動設定的，其目的是爲了在訓練引數的時候讓模型的表現更好。

      What is Data Leakage in Machine Learning?

  • 數據洩露（leakage）：本來不應該出現在X裡的、和目標y有關的數據，出現在了X中。如此一來，機器學習演算法會得到不符合真實的結果。導致：testing過於符合理想，不符合現實
    • 原本要預測未來的房價但是在traindata裡卻出現了未來房價的數據
  1. 測試集數據被洩露到訓練集：模型在現實中的表現遠不如test accuracy；測試集失去意義。
    • 股價
  2. 未來的信息被洩露到過去：時間序列相關，現實中模型將無法有效根據過去情況預測未來。
  3. 第三方信息。例子：已知坐標，利用geocoder類型的服務推出所在城市
